# Bumblebee

A minial fetch for POSIX

## About  

Bumblebee is a simple command line tool that display :   
- OS  
- Shell  
- Terminal  
- Emails  
- Song   


## Prerequisite   

`sudo pacman -S playerctl`  


## Install  


`chmod +x bumblebee`  
`./bumblebee`
